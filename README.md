# jumia_web_scraping
Web scraping project that extracts product data from Jumia using Python, Selenium, and Pandas.
Jumia Web Scraping Project
Overview

This project is a Python-based web scraper that extracts product data from Jumia. It uses Selenium to handle dynamic content and collects information such as product name, price, rating, and image URL.

## Features
Scrapes multiple pages (pagination)
Extracts product name, price, rating, and image
Handles dynamic websites using Selenium
Stores data in CSV format
Displays data in table format using Pandas

## Tech Stack
Python
Selenium
Pandas


## How to Run
Clone the repository
git clone https://github.com/Deepika4456/jumia_web_scraping

## Project Workflow

1. Open Jumia website using Selenium  
2. Load dynamic content  
3. Extract product details  
4. Store data using Pandas  
5. Export to CSV file

## Use Case

This project can be used for:
- Price comparison analysis  
- Market trend analysis  
- E-commerce data research
